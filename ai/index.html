1.Introduction

Third-year BSc student in Artificial Intelligence at Utrecht University, with a prior year in Applied Physics at TU Delft (2022–2023).

Research topic: social sycophancy in large language models, as defined and measured by the ELEPHANT benchmark.[2] The question is why model outputs shift toward perceived user preferences rather than factual accuracy, with implications for NLP, HCI, and AI safety.

This study extends prior work by comparing model behaviour across two deployment contexts: a consumer web interface and a direct API endpoint. The hypothesis is that web interfaces amplify sycophantic tendencies through hidden factors such as system-prompt modifications and interface framing, which are absent in raw API behaviour.

Methodology: fully automated web-interaction pipeline comparing model behaviour across UI and API environments. See github.com/Alek-Klem-UU/SycoUI.
